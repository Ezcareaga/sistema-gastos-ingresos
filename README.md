# Sistema de Control de Gastos e Ingresos

Sistema contable para registro y categorización de transacciones financieras.

## Tecnologías
- Java 17/21
- Angular
- JHipster
- Docker

## Descripción
Aplicación para gestión de:
- Transacciones (ingresos/gastos)
- Categorías contables
- Cuentas
- Contactos/Terceros

## Instalaciones Realizadas

### Backend
- Java 21 (SDKMAN)
- Maven 3.9.11

### Frontend
- Node.js LTS (NVM)
- Angular CLI

### Contenedores
- Docker Desktop + WSL2
